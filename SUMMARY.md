## Workshop Goals

- Introduce participants to version control concepts using **Git**
- Familiarize participants with **GitHub** for collaborative development
- Practice core Git workflows:
  - `git clone` — cloning a repository
  - `git add` / `git commit` — staging and committing changes
  - `git push` — pushing changes to a remote repository
  - Opening and reviewing **Pull Requests**
